# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-14

### Added
- Album art display
- Track info (title, artist, album)
- Playback controls (previous, play/pause, next)
- Seek bar with elapsed/total time
- Volume control slider
- Lyrics view via LRClib (click album art to toggle)
- Spotify Canvas animated video background
- Responsive layout — vertical and horizontal mode
- Pywal integration for automatic color theming
- Spotify Web API support via spotipy
- playerctl fallback (no credentials required)
